# WeChat QR Code Setup

## 📱 添加微信二维码图片

请将微信二维码图片保存为 `wechat-qr.jpg` 并放置在此文件夹中。

### 步骤：

1. **保存二维码图片**
   - 从微信获取您的二维码图片
   - 将图片命名为：`wechat-qr.jpg`
   - 放置在 `images/` 文件夹中

2. **推荐规格**
   - 格式：JPG 或 PNG
   - 尺寸：建议 500x500px 到 1000x1000px
   - 文件大小：< 500KB

3. **测试**
   - 打开网站
   - 点击页脚的微信图标
   - 确认二维码弹窗正确显示

### 功能说明：

✅ 点击微信图标弹出二维码
✅ 点击关闭按钮或外部区域关闭
✅ 按ESC键关闭
✅ 带有淡入动画效果
✅ 响应式设计，适配手机
